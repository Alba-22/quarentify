Bem-vindo(a) ao Quarentify

Quarentify é uma aplicação que usa informações do Spotify para descobrir quais músicas, artistas e gêneros você mais tem ouvido durante esses tempos de pandemia, além de criar uma playlist especial para você. Para começar, clique no botão acima para acessar o Quarentify.
